# Example Entity

> Seed page. Replace with a real person, org, product, place, or work once the
> vault has its first sources. Delete this file when no longer needed.

An entity page collects what the wiki knows about one named thing, with a
citation to every source that mentions it. This seed demonstrates the
[Open Knowledge Format](../index.md) frontmatter every vault page carries:
`type`, `title`, `name`, optional `entity_kind`, `domain`, and `tags`.
